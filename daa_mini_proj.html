Dijkstra's Algorithm

Greedy · Shortest Path · Graph Theory

Finds the minimum-cost route between two cities in a weighted graph by always expanding the cheapest unvisited node first.

  • 1 Set all distances to ∞; source = 0
  • 2 Pick unvisited node with minimum distance
  • 3 Relax neighbours — update if shorter path found
  • 4 Mark visited; repeat until destination reached
  • 5 Traceback predecessor map to get full path
Time: O(V²) · Space: O(V) · Optimal on non-negative weights

0/1 Knapsack Algorithm

Dynamic Programming · Combinatorial Optimization

Picks the most profitable subset of packages within the truck's weight limit. Each item is fully taken or fully left.

  • 1 Build dp[i][w] = max profit with i items, capacity w
  • 2 If item weight > w → dp[i][w] = dp[i-1][w]
  • 3 Else → dp[i][w] = max(skip, take + dp[i-1][w-wᵢ])
  • 4 Backtrack from dp[n][W] to find selected items
  • 5 Item i taken when dp[i][w] ≠ dp[i-1][w]
Time: O(n·W) · Space: O(n·W) · Exact solution via bottom-up DP
